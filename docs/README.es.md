# Farito (Español)

[English](../README.md) · [中文](README.zh-CN.md)

Farito viene de `faro` + `-ito`: un "pequeño faro".

Tu navegador es un mar, y cada página web es una isla.  
Farito es ese pequeño faro a tu lado: discreto, preciso y siempre encendido.

## Descripción

Farito es una extensión Chrome MV3 tipo browser-agent que:
- entiende estructura y contexto de la página (DOM / texto / estado runtime)
- ejecuta acciones bajo políticas de seguridad
- completa flujos multi-paso desde una interfaz de chat

## Puntos fuertes

- UX centrada en chat (Side Panel)
- Trazas en vivo de llamadas a herramientas
- Aprobaciones dentro del chat (Allow / Reject)
- Modo Watcher para automatización proactiva
- Gestión de sesiones (crear/cambiar/borrar/limpiar)

## Etiquetas técnicas

- Manifest V3
- Service Worker
- Content Scripts + Page Bridge
- Tool Calling Agent Loop
- Site Strategy Layer
- Providers compatibles con OpenAI / Anthropic / Zhipu / Z.AI
