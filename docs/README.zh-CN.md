# Farito（中文）

[English](../README.md) · [Español](README.es.md)

Farito 来自西班牙语：`faro`（灯塔）+ `-ito`（小小的），意为「小灯塔」。

你的浏览器是一片海，每个网页是一座岛。  
Farito 是你身边那座小小的灯塔，不喧哗，但能照亮页面每个关键角落。

## 项目简介

Farito 是一个 Chrome MV3 浏览器 Agent 插件：
- 能看懂页面结构（DOM / 文本 / 运行态）
- 能在策略控制下执行动作（点击、输入、提取、导航）
- 通过聊天界面完成多步骤自动化

## 亮点

- 聊天优先交互（Side Panel）
- 实时工具调用轨迹与调试日志
- 审批流内联到对话（Allow / Reject）
- 主动感知 Watcher 模式
- 会话管理（新建/切换/删除/清空）

## 技术标签

- Manifest V3

- Content Scripts + Page Bridge

- Site Strategy Layer
